# User Story: s1.2 - App Bootstrap (Global Prefetch)

**As a** User
**I want** the application to preload all reference data when I open the Dashboard
**So that** subsequent navigation to any entity view has zero latency.

## Acceptance Criteria
- A single backend endpoint `getAppBootstrapPayload()` retrieves all data defined in `APP_SCHEMAS`.
- The data is sanitized using `JSON.parse(JSON.stringify())` to prevent proxy object issues.
- The frontend shows a discrete loading indicator (`<ion-toast>` or `<ion-progressBar>`) during the initial fetch.
- On success, the frontend populates `window.__APP_CACHE__` with all entities.
- Navigation functions (`showListSidebar` / `renderDataView`) use this cache and avoid network requests if data is present.
