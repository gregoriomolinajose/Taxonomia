# Story S1.5: Universal Bulk Data Engine (CSV Import/Export)

## User Story
As a Product Owner,
I want a universal Bulk Import/Export component that works automatically for ALL CRUDs,
So that I can easily migrate and manage large datasets without writing entity-specific code.

## Acceptance Criteria
- Export button in list view downloads current entity cache to CSV.
- Import button reads local CSV and converts to JSON.
- Confirmation modal before importing.
- Backend bulkInsert adds records in one operation with audit columns.
