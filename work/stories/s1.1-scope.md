# Scope S1.1: Agregar saludo de usuario

## In scope:
- Obtener el nombre del usuario activo (o su correo).
- Mostrar el saludo en la interfaz principal tras el login o al abrir la app.

## Out of scope:
- Pantallas de configuración de perfil.
- Edición de nombre de usuario en base de datos.
- Gestión de avatares de usuario.

## Done when:
- El saludo es consistentemente visible en la UI principal.
- El nombre del usuario se recupera de manera segura (ej. desde el backend/sesión local).
- No hay errores en consola si el nombre falta (fallback a "Usuario").
