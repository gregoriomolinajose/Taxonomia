# Scope: S1.3 Blueprint V2 Refactoring & Persona Provisioning

## Boundaries

**In scope:**
- Migration of `Portafolio` schema to V2 (`fields` array format).
- Migration of `Dominio` schema to V2.
- Migration of `Capacidad` schema to V2.
- Migration of `Unidad_Negocio` schema to V2.
- Validation and testing of the `Persona` schema (already in V2) but ensuring backend UI mappings are correct.
- Adapting any associated lookup queries in backend if necessary.

**Out of scope:**
- Modifying UI HTML templates (the UI should automatically adapt to the valid V2 array payload).
- Implementing new Views or Pages in the frontend.
