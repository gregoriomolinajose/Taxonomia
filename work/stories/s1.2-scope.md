# Scope: s1.2 - App Bootstrap (Global Prefetch)

## In Scope
- Creation of `getAppBootstrapPayload()` in `API_Universal.gs`.
- Modification of Dashboard initialization in `Index.html` to call this endpoint.
- Updating `window.__APP_CACHE__` in the frontend dashboard.
- Ensuring `showListSidebar` and `renderDataView` prioritize cached data without refetching if present.

## Out of Scope
- Implementing caching for individual record updates (already handled by S1.1).
- Offline capabilities (IndexedDB/LocalStorage are out of scope for now, just memory cache).

## Done When
- A user loading the app sees the synchronization message ('Sincronizando plataforma...').
- Navigating to any entity for the first time takes ~0.0s latency.
