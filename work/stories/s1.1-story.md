# Story S1.1: Agregar saludo de usuario

## User Story (Connextra)
Como usuario
Quiero ver un saludo personalizado
Para sentirme bienvenido en la aplicación

## Acceptance Criteria (Gherkin)
Escenario: Saludo inicial exitoso
  Dado que el usuario inicia sesión correctamente
  Cuando accede a la página principal
  Entonces debe ver un mensaje de saludo "Hola, [Nombre]"
