# Story s1.4: Sidebar Floating Card

**As a** User
**I want** the sidebar menu to look like a floating card
**So that** the UI has better depth and a modernized look aligned with our CSS tokens.

## Acceptance Criteria

1. The `ion-menu` container uses `var(--rounded-md)` border radius.
2. The menu has `var(--spacing-4)` margins on all sides.
3. The menu uses `var(--shadow-floating)`.
4. In Dark Mode, the floating shadow is subtly adjusted for contrast.
5. In `.shell-mini` state, margins are preserved.
6. Only CSS Design System tokens are used (no hardcoded pixels).
