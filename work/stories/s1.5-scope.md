# Story S1.5: Universal Bulk Data Engine (CSV Import/Export) Scope

## In Scope
- Inject Export/Import buttons in `DataView_UI.html` list view.
- Global `exportCurrentEntityToCSV()` using `window._CACHE` and `APP_SCHEMAS`.
- Global `importCSVToCurrentEntity()` reading file and parsing to JSON array.
- Backend `bulkInsert(entityName, recordsArray)` in `API_Universal.gs` using single `setValues` call.
- Auto-provisioning and audit column injection in backend.

## Out of Scope
- Complex CSV validation beyond basic parsing.
- Partial imports (all or nothing).
- Progress bars for importing extremely huge files (standard loading blocking UI is fine).

## Done When
- Export button generates accurate CSV for active entity.
- Import successfully reads CSV and sends JSON.
- AppsScript successfully imports records to Google Sheets.
