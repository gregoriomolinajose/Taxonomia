# S27.4: UI FormEngine Arquitectura Híbrida (Linear & Stepper)

## Connextra

**Como** desarrollador de la plataforma Taxonomia  
**Quiero** que el `FormRenderer_UI` soporte de forma co-existente renderizado de formularios en 1 solo paso (Linear) o en Multi-Steps (Wizard Stepper) basado en declaraciones topológicas dinámicas.  
**Para** soportar entidades de alta densidad cognitiva mediante wizard con secciones, y mantener la experiencia ágil con Scroll Lineal para entidades simples e hiperdensas de una sola temática, permitiendo integrar divisores en ambas arquitecturas.

## Gherkin AC

### Scenario: Formulario Simple sin Secciones o con Única Sección
**Given** un schema de entidad que posee campos con 0 o 1 `section` diferente  
**When** el `FormRenderer_UI` construye la UI  
**Then** el sistema NO instanciará la interfaz del Wizard (`UI_FormStepper`)  
**And** pintará todas las propiedades en modalidad de listado Linear de un solo Scroll  
**And** el botón por defecto cargado en el footer será "Guardar"
**And** cualquier atributo interno `type: "divider"` será visible y separará conjuntos de campos de forma transparente.

### Scenario: Formulario Complejo Multi-Sección
**Given** un schema de entidad con campos asignados a 2 o más arreglos de secciones topológicas (`section: "A"`, `section: "B"`)  
**When** el `FormRenderer_UI` construye la UI  
**Then** instanciará delegadamente la interfaz del Wizard (`UI_FormStepper`) usando los botones de "Siguiente" y "Atrás"  
**And** presentará las páginas y tabs de Sidebar automáticamente  
**And** "Guardar" será el botón activo en la última etapa validada  
**And** divisores internos pueden seguir siendo pintados dentro de cualquier step.

## Specification by Example

| Input (Schema Config) | Output (FormEngine Behavior) | Footer Actions |
|----------------------|--------------------------------|-------------------|
| { fields: sin sections } | Linear Vertical Scroll Grid | [ Guardar ] |
| { fields: 1 sección global } | Linear Vertical Scroll Grid | [ Guardar ] |
| { fields: secc 1, secc 2 } | UI_FormStepper Wizard Paginado | [ Atrás, Siguiente -> Guardar ] |
| { fields: dev_divider + secc 1 } | Linear Vertical Scroll con Divisor | [ Guardar ] |
