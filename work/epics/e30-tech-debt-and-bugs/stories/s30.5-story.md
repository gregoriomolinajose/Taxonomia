# Story S30.5: AST RegExp CSS Minifier

## User Story (Connextra Form)
Como **Product Architect**
Quiero **evaluar e integrar un generador AST (parse5) real o empaquetador CSS (Rollup/Esbuild) en el script NodeJS CLI de despliegue (`deploy.js`)**
Para **prevenir corrupciones visuales globales derivadas de la técnica casera en RegExp que usamos para minificar y anidar estilos durante la compilación de clasp.**

## Acceptance Criteria
- **Scenario**: El despliegue a QA/DEV ejecuta `deploy.js` empaquetando archivos `.css` en la estructura virtual `.html`.
  - **Given** que existen variables CSS nativas y reglas condicionales de tipo `calc()`, `:host`, `@media`, etc.
  - **When** se comprime y embebe el CSS en el archivo App Script global.
  - **Then** el regex rudimentario es reemplazado por una librería que respeta la semántica del CSS sin fusionar atributos ilegalmente ni corromper variables complejas.

## Gherkin AC

