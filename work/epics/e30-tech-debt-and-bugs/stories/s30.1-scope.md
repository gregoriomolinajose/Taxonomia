# Scope: S30.1 Limpieza Segura de Assets Temporales

**In scope:**
- Modificación del script `deploy.js` (remover la fase de mutación `fs.rmSync`).
- Actualización del archivo `.claspignore`.

**Out of scope:**
- Cambiar la tecnología de compilación (esto pertenece a S30.5 AST RegExp CSS Minifier).

**Done when:**
- `deploy.js` ejecuta la preparación del código sin intentar borrar el directorio `.build/assets` de forma forzosa.
- Clasp ignora correctamente archivos CSS y recursos en `.build/assets` nativamente.
