# Story S30.1: Limpieza Segura de Assets Temporales

## User Story
**Como** mantenedor del pipeline de despliegue
**Quiero** delegar la exclusión de archivos temporales al archivo `.claspignore` nativo
**Para** evitar la mutación paralela del File System y el borrado riesgoso de `.build/assets` durante el build.

## Acceptance Criteria
- El script `deploy.js` ya no debe intentar hacer `fs.rmSync` sobre `.build/assets`.
- El archivo `.claspignore` debe estar configurado para ignorar explícitamente `**/*.css` y otros bins generados dentro de `src/` o `.build/`.
- El despliegue (`npm run deploy` o Clasp push) debe continuar enviando únicamente los archivos esenciales (.js, .html, .gs) sin empujar los tokens crudos CSS nativos al servidor.
