# Scope S18.1: Session Topology Resolver

## Objective
Construir el interceptor backend y el inyector que alimenta a la ventana con el grafo matemático del usuario durante el ciclo de vida de carga (`Auth_UI` / `JS_Core`).

## In Scope
- Carga transaccional del grafo de pertenencias.
- Emisión del payload stringificado dentro de Index o en la resolución del Token.
- Parseo estricto del ABAC Object (`window.__ABAC_CONTEXT__`).

## Out of Scope
- Renderizado visual o apagado funcional en las vistas Front-end (Corresponde a S18.4).
- Middleware de intercepción de escritura en Base de datos (Corresponde a S18.2).

## Definition of Done
- Objeto de sesión disponible en la consola mediante `window.__ABAC_CONTEXT__` con la forma `{ ownerOf: [...ids], memberOf: [...ids] }`.
