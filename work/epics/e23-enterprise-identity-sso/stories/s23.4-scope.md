# Story 23.4 Scope

## Objective
Purgar rastros de la Deuda Técnica originada en las iteraciones tempranas (Mocks y logs huérfanos) reforzando una capa de "Estrictez Zero-Trust".

## In Scope
- Eliminar abstracciones MOCK en `src/API_Auth.js`.
- Eliminar `MockEngine` en `src/DataAPI.client.js`.
- Intervención de `console.error` dentro de `src/UI_FormSubmitter.client.js`.

## Out of Scope
- Intervenir otros tests de Jest que sean funcionales (Jest sí debe poder inyectar datos de prueba en la capa mockeada por `.test.js`, pero la capa de API no debe estar agujereada con parámetros).

## Done Criteria
- Taxonomía es incapaz de levantar "falsamente". Si no hay conexión local de Sheets o Apps Script context real, el sistema entra honorablemente en Error de Red en vez de simular exito mediante `MockEngine`.
