# Story 23.1 Scope

## Objective
Implementar un `IdentityController` backend en GAS para usar el SDK de Directorio, sumado a un `AvatarCacheManager` en el UI JS_Core.

## In Scope
- Creación de función backend `getWorkspaceAvatar(email)`.
- Client-side memoized Map (`window.AvatarCache` o dentro de `global`).

## Out of Scope
- Conectividad a bases de datos relacionales.
- Panel UI dinámico para inputs (S23.2).

## Done Criteria
- [ ] Endpoints desplegados y funcionando con Auth de admin.
- [ ] Caché testeada y nulas peticiones duplicadas.
