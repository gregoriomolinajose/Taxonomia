# S17.2: Theme Engine Segregation (Story Design)

## Description
Extraer toda la manipulación, persistencia e inyección del Tema Estético (CSS Vars, localStorage, Theme Cycling) que habita en `JS_Core.html` para mandarlo a `Theme_Engine.html`.

## Bounded Scope
- Mover el objeto literal `window.ThemeManager`.
- Inyectar la inicialización en la cascada de carga nativa en `Index.html` (antes que el EventBus).

## Dependencias
- Requiere acceso al DOM tempranísimo para evitar "Flashazos de Luz" en recargas web.

## Criterios de Cierre
- El Botón `id='theme-toggle-icon'` sigue ciclando los modos oscuro/claro/oceano.
- ApexCharts refleja correctamente los callbacks de Reflow CSS.
