# Architecture Review: S17.1 (Treemap Logic Extraction)

> Evaluando la proporcionalidad y necesidad de los cambios estructurales mediante las Heurísticas de Diseño Simple (Beck's Simple Design Rules).

---

## 1. Auditoría de Responsabilidades (H11: Change Reason Count)
* **Heurística Evaluada:** Antes de la extracción, un simple cambio en el algoritmo matemático O(N) para re-estructurar el diagrama obligaba a modificar `JS_Core.html`, violando el principio SRP, ya que el archivo "Booteador" cambiaba por motivos ajenos a "Bootear".
* **Veredicto:** **Resuelto (PASS).** La mudanza hacia `Math_Engine.js` alinea perfectamente el código con la Regla 2 (No tener razones dobles para cambiar). Ahora el core UI inicializador no necesita recompilarse si cambia un requerimiento de las gráficas jerárquicas D3/ECharts.

## 2. Auditoría de Abstracciones Muertas (H2: Wrapper Without Logic)
* **Heurística Evaluada:** El objeto nativo global `window.Cache_Utils` dentro de `JS_Core.html` funcionaba como un "falso Namespace" que de hecho sólo contenía 1 única función (el generador del treemap).
* **Veredicto:** **Simplificado (PASS).** El falso Namespace se extirpó. La lógica ahora yace en un Export de Módulo de Node (para poder pasarle pruebas Jest Unitarias de forma nativa) o en el Namespace isomórfico genérico de matemáticas.

## 3. Topología de Acoplamiento (H14: Coupling Direction)
* **Heurística Evaluada:** `DomainMap_UI.html` (La Vista especializada) importaba sus datos gráficos desde el Bootstrapper General, generando un acoplamiento donde una Vista dependía del Framework entero para mapear su estructura semántica.
* **Veredicto:** **Mejorado (PASS).** La Vista ahora depende directamente de un motor matemático agnóstico que recibe el contexto estricto local (`window.__APP_CACHE__`) como parámetro funcional puro. Las dependencias ahora fluyen correctamente. 

---

## Veredicto Final del Auditor

- [x] **PASS / SIMPLIFY CORRECTED:** El código final borró deuda algorítmica y no introdujo abstracciones prescindibles. Resulta matemáticamente impecable.
