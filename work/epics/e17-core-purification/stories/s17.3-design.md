# S17.3: Auth Layer Decoupling (Story Design)

## Description
Desmantelar la lógica "destructiva" de Seguridad (`AuthManager`) que actualmente reside en `JS_Core.html`. 
La implementación moderna bajo el estándar Blueprint V4 debe dejar de inyectar/eliminar cadenas crudas de HTML (`innerHTML = ''`) y transicionar a una semántica Declarativa basada en Eventos utilizando Plantillas Nativas (`<template>`) sobre `Auth_UI.html`.

## Bounded Scope
- Mover el objeto literal `window.AuthManager`.
- Definir un `<template id="tmpl-auth-denied">`.
- Usar _display:none_ en lugar de _innerHTML=''_.

## Criterios de Cierre
- El ciclo de validación detiene la app, oculta el dashboard principal mientras carga, y al fallar despliega el template de denegación sin destruir la memoria base del HTML subyacente.
