# User Story
Como desarrollador principal 
Quiero eliminar las advertencias de arquitectura y deuda técnica visual detectadas
Para mantener el sistema alineado con RaiSE Strict (Zero Warnings) y robustecer el DesignSystem.

## Contexto Técnico
- **Lints de CSS (`stylelint`)**: 115 advertencias de severidad por colores estáticos en la capa DOM/CSS (`CSS_App.html` y `assets/css/data-view.css`). Deben mapearse a las variables del sistema (`var(--ion-color-*)` o System Vars).
- **Lints de Arquitectura**: `UI_FormSubmitter.client.js:175` invoca explícitamente a `window.DataAPI.call`, evadiendo la inyección de dependencias (DI).

## Acceptance Criteria
- [ ] Todo hardcoded color detectado por `scale-unlimited/declaration-strict-value` fue refactorizado y reemplazado por la directiva CSS variable correspondiente sin romper el Light/Dark mode.
- [ ] La llamada directa a DataAPI de `UI_FormSubmitter.client.js` fue corregida delegándola por DI.
- [ ] El comando `npm run lint:arch` pasa impecable con Exit Code 0.
- [ ] El comando `npm run lint:css` pasa limpio con Exit Code 0.
