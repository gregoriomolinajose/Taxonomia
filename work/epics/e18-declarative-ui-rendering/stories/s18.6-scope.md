# S18.6 Scope: Root Eradication (Global Nullification)

## Objective
Alcanzar el 100% de purismo arquitectónico V4 erradicando hasta el último vestigio estático del patrón de inyección DOM mediante strings (`.innerHTML`) en toda la aplicación.

## Target Area
1. `UI_Router.html`: Inyección masiva de menú lateral.
2. `DataView_UI.html`: Construcción estática del esqueleto y generador de popovers.
3. `FormBuilder_Inputs.html`: Generación cosmética de botones y Chips iterados.

## Definition of Done
Toda construcción visual que se inyecte en el flujo principal del documento o modales utilizará métodos formales `document.createElement`, `.appendChild()`, y `replaceChildren()`, o instanciará sobre las plantillas `<template>` pasivas existentes.
