# S18.2 Retrospective

**What went well:**
1. Eliminamos sistemáticamente todas las inyecciones estáticas CSS de `.style.display` a lo ancho del proyecto migrando hacia `.classList.toggle('ion-hide')`.
2. El uso masivo de selectores regex automatizados fue seguro y no comprometió el AST transpilado por `deploy.js`.

**What to improve:**
1. Tuvimos un sesgo ciego: olvidar que el HTML estático raíz (`Index.html`) poseía sus propias reglas `style="display:none"` inyectadas duramente, lo que bloqueaba el nuevo JS. Tuvimos que intervenir manualmente el layout original V3 al final del ciclo.

**Action items:**
1. Siempre auditar primero si las "vistas contenedoras madre" (DOM inicial) no tienen código heredado bloqueante (inline styles root) *antes* de comenzar a escribir transformadores semánticos en Javascript puro.
