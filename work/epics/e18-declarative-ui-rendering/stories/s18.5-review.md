## Retrospective

### What went well
- **Eficacia de Ionic Native Nodes:** Crear los `ion-item` e `ion-checkbox` puramente como Node nos permitió interceptar el burbujeo de clics nativos discriminando el Layer objetivo `e.target.tagName !== 'ION-CHECKBOX'`.
- **Garbage Collection Segura:** El modal efímero ahora recoge en sí mismo los listeners atados, mitigando posibles Zombie Memory Leaks detectados teóricamente en auditorías V3.
- **Factoría Reusable V4:** Como subproducto de la investigación de S18.5, comprobamos que `FormBuilder_Inputs` ya gozaba de pureza nodal.

### What needs improvement
- Las configuraciones estáticas para los atributos (`btn.setAttribute()`) dominaban la densidad, sugiriendo a futuro la inversión hacia pequeñas funciones utilitarias *Helper* (e.g. `Element.setAttributes({color: 'primary', shape: 'round'})`) para aligerar líneas de código textuales si la lógica procedural continúa escalando (React-like minimal helpers).
