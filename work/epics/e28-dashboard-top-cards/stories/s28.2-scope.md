# S28.2: Rediseño Visual Contadores

## Objective
Alinear las tarjetas indicadoras del dashboard a la referencia visual (UI Spec). Eliminar los botones de creación en el dashboard para favorecer las acciones dentro de los DataViews, limpiando el layout.

## Functional Changes Tracker
- Eliminados botones Inferiores (`Crear Nuevo`, `Ver Listado`).
- Reemplazo de grid header por layout Flexbox (Banda de color Izquierda / Icono / Divisor / Textos).
- Adopción absoluta de variables CSS nativas para el Dark Mode en vez de colores forzados.
- Se asegura que los colores en la metadata de cada entidad hidraten visualmente el icono y la banda del dashboard.
