# S28.1 Backend Counters API

**Como** Área de Negocio
**Quiero** que el backend proporcione un endpoint rápido para sumar los registros de las entidades fundamentales
**Para** no tener que descargar toda la tabla en el navegador y evitar saturar la memoria de la interfaz principal.

## Acceptance Criteria
- Se integra un endpoint específico (ej. `action === 'getDashboardCounters'`) en el backend que sirva los conteos de Portafolio, Equipo y Persona.
- Este endpoint reutiliza la caché en memoria de `Engine_DB.list` (`CACHE_LIST_*`) para realizar conteos pasivos instantáneos, logrando un performance predecible.
- Devuelve la suma como un payload JSON de formato simple (Status success y objeto de contadores).
