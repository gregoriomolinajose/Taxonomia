# Scope: S28.1 Backend Counters API

## In Scope
- Agregar comportamiento en el enrutamiento de Apps Script (ej. `API_Universal.gs` o un equivalente) para devolver los recuentos rápidos estadísticos.
- Implementar la sumatoria respetando el estado lógico (sin contar registros con `estado: 'Eliminado'` si son provistos por el Engine_DB).

## Out Scope
- Renderizado visual ni maquetación UI (Delegado estrictamente a S28.2).
- Modificaciones sobre la base de datos o esquemas.

## Done Criteria
- Llamar al controlador mediante `google.script.run` o `DataAPI` retorna `{"Portafolio": x, "Equipo": y, "Persona": z}` exitosamente en consola frontal.
- Los tiempos de respuesta no muestran penalizaciones I/O excesivas porque se apoya en Caché LRU del Servidor.
