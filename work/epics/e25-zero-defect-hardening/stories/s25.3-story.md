# Story S25.3: Restauración de Interacciones de DataGrid

## User Story

**Como** analista de datos / administrador de dominios
**Quiero** hacer click en las filas de la tabla de datos principal (DataGrid) y seleccionar nodos 
**Para** poder abrir el drawer en modo edición, ver el desglose de relaciones SCD-2 interactuar fluidamente sin recargar la página o perder dependencias de DOM.

## Acceptance Criteria

### Gherkin Scenarios

```gherkin
Feature: DataGrid Record Selection
  Scenario: Editing an existing record
    Given the DataGrid is populated with Domain Entities
    When the user clicks on a table row
    Then the system should capture the Record ID
    And open the Mobile First Drawer in "edit" mode loaded with the node's payload
    And the DataGrid's internal DOM structure must not be hijacked or corrupted by the interaction.
```
