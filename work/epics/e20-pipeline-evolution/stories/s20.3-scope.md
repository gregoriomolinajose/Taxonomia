# Scope S20.3: ES6 ThemeManager Evolution

## In Scope
- Refactorización de `UI_ThemeManager.html` usando el Patrón de Módulo de JavaScript (Orquestador).
- Encapsulación de funciones globales (`hydrateThemeConfig`) como métodos del módulo (`ThemeManager.init`, `ThemeManager._hydrate`).
- Remoción de ejecución procedural script-level en el archivo the manager.

## Out of Scope
- Migración a Webpack o Rollup. Usaremos ES5/ES6 simple mediante objeto para no re-escribir el compilador ahora mismo.
- Cambios a las paletas de colores subyacentes o valores hex.

## Done Criteria
1. El archivo expone un solo objeto `window.ThemeManager`.
2. Las llamadas ocurren orquestadamente cuando el Bootstrapper detona `ThemeManager.init()`.
