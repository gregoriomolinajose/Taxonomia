# Story Plan: S20.2 - Arquitectura de Asset Management

## Overview
- **Story ID:** S20.2
- **Size:** S (3-5 SP)
- **Goal:** Aislar limpiamente el manejo de configuraciones UI y de assets visuales lejos de las capas de servidor o de pipelines inyectores.

## Tasks

### T1. Setup Frontend Context (`JS_Core.html`)
- **Descripción:** Definir localmente el objeto global de UI con el respectivo diccionario de colores `badgeMap`.
- **Archivos:** `src/JS_Core.html`
- **Verificación:** Existencia en memoria del navegador de `window._UI_CONFIG`.
- **Dependencias:** Ninguna.

### T2. Backend UI Purge (`Schema_Engine.gs`)
- **Descripción:** Borrar el nodo `_UI_CONFIG` obsoleto dentro de la declaración de tabla base `APP_SCHEMAS`.
- **Archivos:** `src/Schema_Engine.gs`
- **Verificación:** Ejecutar la extracción de JSON sin que `_UI_CONFIG` aparezca en el payload de Gas.
- **Dependencias:** T1.

### T3. Refactor Consumidor (`UI_DataGrid.html` & `Governance_Admin_UI.html`)
- **Descripción:** Cambiar la lectura de fallback `window.APP_SCHEMAS._UI_CONFIG.badgeMap` a `window._UI_CONFIG.badgeMap` en ambos grids.
- **Archivos:** `src/UI_DataGrid.html`, `src/Governance_Admin_UI.html`
- **Verificación:** Recarga limpia del renderizado Web sin que las Badges rompan su color asignado.
- **Dependencias:** T1, T2.

### T4. Limpieza del Orquestador (`deploy.js`)
- **Descripción:** El bloque donde detectamos y borramos físicamente `assets/css` previo a subir el `.build` será eliminado. Confiaremos pasivamente en `.claspignore`.
- **Archivos:** `deploy.js`
- **Verificación:** Ejecutar `node deploy.js dev` para confirmar que no invoca una ruta fallida internamente ni arroja Error.
- **Dependencias:** Ninguna (Independiente).

## Execution Order
1. **T1 -> T2 -> T3** (Secuencia Core Configuración - Requiere consistencia síncrona).
2. **T4** (Paralelo / Independiente - Infraestructura Pipeline).

## Risks & Mitigations
- **Riesgo:** Posible desincronización del badgeMap si hay componentes externos sin remapear.
- **Mitigación:** Asegurar que un grep masivo `_UI_CONFIG` en el `src` no halle nada omitido. (Anteriormente ejecutado, solo existía en DataGrid y GovernanceUI).
