## What went well / What to improve
- **Well**: La migración a Object Literal (`window.ThemeManager`) fue limpia y precisa, integrándose perfectamente al ecosistema asíncrono actual.
- **Improve**: Mantener disciplina sobre evitar scripts libres ("floating scripts") flotando debajo de las definiciones en archivos HTML; desde el primer instante que el equipo detecte procedimentalidad agresiva se debe encerrar.

## Heutagogical Checkpoint
1. **What did you learn?** Re-afirmo la importancia de que todo archivo frontend debe abstenerse de manipular el DOM en tiempo de declaración. 
2. **What would you change about the process?** Añadiría una alerta estática linter para scripts fuera de Hooks en el pipeline futuro.
3. **Are there improvements for the framework?** La heurística H2 (Isolated Lifecycles) nos sirvió eficientemente para acordonar esta vulnerabilidad en este proyecto mixto. 
4. **What are you more capable of now?** Podemos manejar Módulos nativos Vanilla en entornos híbridos Google Apps Script donde no existen build-pipes complejos, maximizando compatibilidad y mitigando errores de colisión.

## Improvements Applied
- Prevención total de "Flash of Unstyled Data" o colapsos de dependencias inversas porque ahora el ThemeManager despierta dócilmente cuando Core se lo ordena.
