# Scope
Desarrollar la interfaz base (Frontend) del Hub de Ingesta Inteligente basada en modales nativos de Ionic. El Modal aislará las múltiples herramientas (Cloud vs Local) para facilitar una UX limpia y escalable.

## Requirements
- El Modal debe permitir el ingreso de una URL pura de Google Sheets.
- Debe mantener visibles funcionalmente el componente de importación "viejo" (Input Type File de CSV) como alternativa Legacy.
- Se montará sobre las APIs inyectadas por `DataView_UI`.

## Done Criteria
- Cuando el usuario presiona "Importar CSV" en la Toolbar, en lugar de lanzar la ventana de SO (Archivos), se mostrará un elegante Modal.
- La inserción o selección pasará callbacks hacia `DataEngine_UI` (S38.3 se colgará después).
