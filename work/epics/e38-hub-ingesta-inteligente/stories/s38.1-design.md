# Arquitectura Visual: S38.1 - Componente Hub Modal ETL

## Acercamiento
Dado el incremento en responsabilidades (Plantillas Cloud, Extracción Directa desde Drive, Modos CSV Locales), la antigua delegación directa a un `<input type="file">` sobrecarga cognitivamente el DOM de la Toolbar.

Se introducirá un **Módulo Puro de UI** (`UI_ETL_Modal.client.js`) que:
1. Funciona bajo el patrón *Stateless Presentational Component*, recibiendo delegados funcionales (callbacks) para que sea agnóstico de su lógica subyacente.
2. Contiene 2 "Bloques Visuales" usando Ionic Cards o un Layout segmentado:
   - **Ingesta Híbrida Cloud:**
     - Botón "Generar/Cargar Plantilla Nueva en Drive".
     - Ionic Input `type="url"` para ingresar un Enlace de Hoja.
     - Botón "Sincronizar vía Internet".
   - **Ingesta Local Backup:**
     - Botón "Subir archivo.CSV plano".
