# Scope: S19.3 (AppEventBus Telemetry Foundation)

## Boundaries

### In Scope
- Creation of `window.Telemetry` (PubSub singleton).
- Log routing logic inside the module.
- Formatting levels (INFO, WARN, ERROR, FATAL, METRIC).

### Out of Scope
- Actually hooking up Telemetry to the Forms and Grid! (We just build the foundation here).
- Connecting Google Analytics or AppsScript HTTP Requests. Focus purely on the PubSub/Console bus for now to minimize quota burning.

## Done Criteria
- [ ] Objeto `Telemetry` inicializa exitosamente en `window` al cargar `JS_Core`.
- [ ] Ejecutar `Telemetry.track('test')` imprime un JSON estructurado de forma asíncrona sin frenar el debugger inmediato.
