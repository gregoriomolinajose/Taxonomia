# S19.5: Telemetry Payload Sanitization

## User Story
Como Administrador de Datos,
Quiero que los payloads enviados a `Telemetry.track` sean sanitizados (truncando referencias circulares, elementos DOM crudos y excesos de profundidad),
Para evitar errores de "Maximum Call Stack Size Exceeded" y saturación de la memoria al enviar a AppEventBus.

## Acceptance Criteria
- [ ] Implementar un método estático como `Telemetry.sanitize(obj)` en `JS_Core.html`.
- [ ] Eliminar referencias circulares de forma segura.
- [ ] Evitar serializar Elementos nativos del DOM.
- [ ] Añadir JSDoc firmando el uso correcto en `window.Telemetry`.
