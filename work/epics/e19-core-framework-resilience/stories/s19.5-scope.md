# Scope: S19.5 (Telemetry Payload Sanitization)

## Boundaries

### In Scope
- Creación de un helper privado/estático `window.Telemetry._sanitize(payload)` en `JS_Core.html`.

- Lógica de ofuscación de contraseñas de red (PII removal) dado que es capa local.
