# Story Design S37.2: DataGrid Reactivity & Formato Semántico

## Qué & Por Qué
- **Problema**: El DataGrid actual (`UI_DataGrid.client.js`) expone en la interfaz Identificadores Léxicos u obscurecidos (UUIDs). Además, ante actualizaciones de estado (por ejemplo, subgrids interactuando de hijos), la vista no se redibuja, requiriendo un *F5* agresivo manual de parte del usuario, lo que corrompe la sensación Single Page Application (SPA).
- **Valor**: La resolución inteligente oculta claves del servidor humanizando la grilla, y el orquestador UI reacciona y refresca al vuelo cada cambio que la capa `DataStore` confirma.

## Enfoque de Solución
Inyectar un Event Listener sobre `AppEventBus` hacia `DATA::UPDATED`.
Usar una técnica de `requestAnimationFrame` para agrupar mútiples eventos de mutación generados simultáneamente y re-renderizar la grilla de datos solo 1 vez para optimizar rendimiento frente a un Redraw masivo (Thrashing).

### Componentes Afectados
- `DataView_UI.client.js` [MODIFICAR]: Controlador global de vista, inyectando el orquestador de colas de refresco (Queue Redraw).
- `UI_DataGrid.client.js` [MODIFICAR]: Capa visual, refactorizando a un forzado duro del ordenamiento de columnas y transmutando semánticamente (Logical Value).

## Ejemplos Concretos (AC Tests)

```javascript
// La grilla capta el evento global y planifica el redibujo visual en el siguiente frame sin asfixiar la UI
window.AppEventBus.subscribe('DATA::UPDATED', () => {
   if (!this._isRedrawQueued) {
       this._isRedrawQueued = true;
       window.requestAnimationFrame(() => {
          this._rerenderData();
          this._isRedrawQueued = false;
       });
   }
});
```

## Acceptance Criteria
- **MUST**: Los IDs internos (UUID / id_registro) deben convertirse en el valor definido en `labelField`, `titleField` o `nombre`.
- **MUST**: Las vistas mutan automáticamente escuchando el nodo `DATA::UPDATED`.
- **MUST NOT**: Utilizar llamadas ciegas directas `_rerenderData();` sin debouncing o encolamiento, para evitar degradar el hardware cliente si hay N mutaciones paralelas.
