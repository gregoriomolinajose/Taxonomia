# S37.6 Scope & Done Criteria

## 1. Boundaries

**In Scope:**
- Extracción de la sección Título/Etiqueta Superior/Ícono y Botón "Cerrar" del Drawer a un componente reutilizable.
- Creación de un método genérico para instanciarlos en `FormRenderer_UI`, `DataView_UI` u otro módulo similar.
- Actualización de los módulos existentes para invocar este nuevo `UI_Factory` o constructor similar.
- Refactorización de estilos globales en `index.css` si están acoplados.

**Out of Scope:**
- Implementación de estado (El Header debe ser puramente presentacional).
- Rediseño mayor del Header (Mismas características estéticas Atlassian pero limpias y reusables).

## 2. Done Criteria

- [ ] Las clases redundantes de Header en `FormRenderer_UI` se han eliminado y se utiliza la nueva aproximación genérica.
- [ ] La UI pasa inspección visual sin regresiones tipográficas o de color.
- [ ] La suite automatizada de UI Tests (`__tests__/*.ui.test.js`) en Chromium verifica la interoperabilidad del Header y sus botones de cierre.
