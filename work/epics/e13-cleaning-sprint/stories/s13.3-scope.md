# S13.3 Scope: Legacy Alias Purging

## Goal
Eliminar las funciones envoltorio (wrappers) retrocompatibles de enrutamiento que ensucian el Scope global de `window`, forzando el uso estricto y explícito del namespace `window.UI_Router`. 

## Done Criteria
- [ ] Eliminar `window.navigateTo`, `window.renderMainNav`, `window.showHomeSidebar`, `window.showListSidebar`, `window.returnToDashboard` y `window.showFormSidebar` de `UI_Router.html`.
- [ ] Refactorizar todas las llamadas en `Index.html`, `JS_Core.html`, `FormRenderer_UI.html` y `DataView_UI.html` para invocar métodos directamente del objeto `UI_Router`.
