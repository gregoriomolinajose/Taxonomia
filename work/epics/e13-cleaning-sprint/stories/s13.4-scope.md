# S13.4 Scope: CSS AST Minifier

## Goal
Implementar un empaquetador CSS verdadero, basado en *Abstract Syntax Tree (AST)*, para reemplazar las expresiones regulares frágiles que se usan dentro de `deploy.js` durante la construcción.

## Done Criteria
- [ ] Eliminar la lógica Regex que rompe los bloques de `calc()` y variables locales de CSS (`CSS_App.html` y `CSS_DesignSystem.html`).
- [ ] Instalar la dependencia de Node (ej. `clean-css`) como paquete de desarrollo.
- [ ] Invocar el AST Minifier de forma segura dentro del Pipeline de `deploy.js` emitiendo reportes de error visibles si los estilos están corruptos.
