# User Story: S5.5

## Connextra
**Como** Analista del Dashboard Ejecutivo
**Quiero** visualizar la topología de la taxonomía (Treemap ECharts) estrictamente basada en las Aristas Activas (`es_version_actual = true`) y de dependencia jerárquica primaria (`tipo=Militar_Directa`)
**Para** evitar la aparición de nodos fantasmas históricos (SCD-2) o enlaces cruzados transversales que formen ciclos quebrando la visualización interactiva del Apache ECharts.

## Gherkin ACs
```gherkin
Feature: Temporal ECharts Graph Rendering
  
  Scenario: Filtrado Dinámico de Grafos Temporales
    Given el Caching System Frontend `__APP_CACHE__` posee matrices de `Relacion_Dominios` donde coexisten aristas activas y caducadas
    When el usuario abre la vista `DomainMap_UI.html`
    Then la subrutina `buildEChartsTreemapData` intercepta el array de relaciones
    And descarta todas las iteraciones donde `es_version_actual` sea estrictamente `false`
    And descarta los tipos de relaciones ajenas a "Militar_Directa"
    And ensambla el modelo jerárquico purgado enviándolo al renderizado gráfico O(N).
```
