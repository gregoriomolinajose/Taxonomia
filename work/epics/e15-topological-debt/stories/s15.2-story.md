# Story: S15.2 (Purge V8 Global Leaks & GC)

## Context
Various foundational components (`FormStepper`, `FormUtils`, `ModalManager`) from Epics 11 and 14 exposed their internal methods to the raw `window` object to allow seamless legacy integrations. Now that the SPA is fully modularized via ES6 rules, these global aliases present memory leaks and severe pollution risks.
